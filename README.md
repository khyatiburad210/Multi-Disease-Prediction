# Multi-Disease Prediction System

## 📌 Description
This project predicts diseases like diabetes and heart disease using Machine Learning algorithms.

## 🛠 Technologies Used
- Python
- Machine Learning
- Pandas, NumPy
- Scikit-learn

## 📂 Dataset
- Diabetes dataset
- Heart disease dataset

## ⚙️ Models Used
- Support Vector Machine (SVM)
- Logistic Regression

## 🚀 How to Run
- Open the .ipynb files in Jupyter Notebook or Google Colab
- Run all cells to see predictions
